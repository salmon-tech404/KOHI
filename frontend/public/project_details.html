03
Backend API & Kết nối
Hoàn thành
Models
Product model — name, description, price, category (ObjectId ref Category), image, available
Order model — customerName, phone, pickupTime, note, items[], total, status (pending/confirmed/ready/done/cancelled)
Category model — name (unique, trim, timestamps)
Admin model — username, password (bcrypt hash pre-save hook), comparePassword method
API Routes
GET /api/products — lấy sản phẩm available, .populate("category"). ?all=true trả hết (admin)
POST /PUT /DELETE /api/products — CRUD sản phẩm (requireAuth)
GET /api/categories — danh sách category (public, sort alpha)
POST /DELETE /api/categories — tạo/xóa category (requireAuth), bắt lỗi duplicate 11000
POST /api/orders — tạo đơn hàng mới (public)
GET /api/orders — tất cả đơn hàng cho admin (public GET, sort createdAt desc)
GET /api/orders/:id — tra đơn theo ID (public)
PATCH /api/orders/:id/status — đổi status (requireAuth)
PATCH /api/orders/:id/cancel — khách hủy đơn (public, chỉ pending)
POST /api/admin/login — xác thực, trả JWT (8h)
Frontend API Layer
api/axios.js — axios instance, baseURL từ import.meta.env.VITE_API_URL
Interceptor tự xóa token và redirect /admin/login khi nhận 401
api/orders.js — createOrder, cancelOrder, getOrderById
api/admin.js — loginAdmin, getOrders, updateOrderStatus, CRUD products, getCategories, createCategory
hooks/useProducts.js — fetch sản phẩm từ API
hooks/useCategories.js — fetch categories từ API
utils/orderStorage.js — saveOrderId / getOrderIds (localStorage key "kohi_orders")
Seed Data
seedCategory.js — tạo 3 categories (Espresso, Cold Brew, Non-Coffee) + 10 sản phẩm với ObjectId refs
npm run seedCategory — chạy từ thư mục backend
seedAdmin.js — tạo admin account (username: admin / password: adminkohi2026)
npm run seed:admin — chạy từ thư mục backend

Admin Dashboard
Hoàn thành
Auth ✓
Admin model — bcrypt pre-save hook (Mongoose 7+ async, không dùng next)
comparePassword dùng bcrypt.compare(plain, hash)
POST /api/admin/login — trả JWT ký bằng process.env.JWT_SECRET (8h)
middleware/auth.js — requireAuth: verify JWT, gắn req.adminId, trả 401 nếu sai
hooks/useAdminAuth.js — getToken / setToken / removeToken / isAuthenticated
localStorage key: "kohi_admin_token"
AdminRoute.jsx — Outlet pattern, redirect /admin/login nếu !isAuthenticated()
Trang /admin/login — form, lưu token, redirect /admin/orders
axios interceptor — bắt 401 mọi API call, removeToken() + redirect login tự động
Layout ✓
Dashboard.jsx — sidebar (NavLink active=gold) + Outlet + nút Đăng xuất
Sidebar nav: Đơn hàng (/admin/orders) · Sản phẩm (/admin/products)
Quản lý đơn hàng ✓
Trang /admin/orders — danh sách dạng hàng ngang, compact
Cột: ngày giờ · tên · SĐT · món · giờ lấy · tổng · status badge · dropdown đổi status

Dropdown đổi status: pending → confirmed → ready → done (không có cancelled)
Đơn đã hủy — dropdown ẩn, hiện dấu "—", không thể đổi status
Status colors: pending=yellow · confirmed=blue · ready=orange · done=green · cancelled=red

Mỗi sản phẩm: ảnh thumbnail + tên + mô tả truncate + giá + badge Ẩn + nút Sửa/Xóa
Modal form thêm/sửa — tên, mô tả, giá, URL ảnh, category dropdown, checkbox ẩn/hiện

Thông báo & Security
Sắp tới

Input validation — sanitize dữ liệu (express-validator)
